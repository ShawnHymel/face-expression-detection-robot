On the UNO Q board, copy *models/model.eim* to */home/arduino/.arduino-bricks/ei-models/model.eim*. Copy this project to *~/ArduinoApps/inference-test-ei*. Run with:

```sh
arduino-app-cli app start ~/ArduinoApps/inference-test-ei
```
